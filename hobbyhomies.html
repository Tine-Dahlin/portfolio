Hobby Homies

3.semester exam - group project

· Developed with HTML5, CSS3 and JavaScript
· Mobile device: Iphone 6/7/8 plus
· Single Page Application (SPA)
· Firebase for login and data storing
· Wireframes and mockups designed in Adobe XD

Hobby homies is a web app where you can socialize with new people who cultivate the same interests as you. It's a web app where you can participate in events and group chats depending on what your hobbies are. The user has the possibility to create events themselves, and let other members join the activity for a meetup. This could be a few people meeting for a game of football or a couple of beers.

With this web app we wanted to create a simple platform, with the opportunity to interact with other people without hassle. It should be easy for the users to create and attend meetups. By creating this, we make it possible for the users to expand their social circle in a simple and appealing way.
